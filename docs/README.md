# Technical Challenge – Sourcing Analysis with AI

## Objective

Analyze sourcing and recruitment data to identify conversion patterns, evaluate sourcing effectiveness, and generate actionable recommendations for recruiters.

## Deliverables

### Documentation

The project includes a technical report describing:

* Analytical approach
* Key decisions made
* AI implementation
* Main findings and recommendations

### Presentation

An executive presentation was created to summarize:

* Recruitment funnel performance
* Sourcing channel effectiveness
* Recruiter performance patterns
* Candidate success indicators
* AI-assisted recruiting recommendations

## AI Usage

AI was used to support hypothesis generation, insight exploration, and the design of a recruiter copilot concept that transforms analytical findings into actionable recommendations.
